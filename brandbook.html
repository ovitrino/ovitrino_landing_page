01 / Overview

Brand Overview

Infrastructure for collectors, clubs, and institutions to manage collections, showcase assets, and conduct private sales in closed rooms — on your terms, outside public marketplaces.

Values

Privacy

Every transaction, every collection, every conversation stays within the room. No public exposure unless the owner decides otherwise.

Control

Owners set the rules. Access, visibility, pricing, and terms are determined by the room creator, not the platform.

Trust

Verified participants, transparent provenance, and secure infrastructure ensure every interaction is built on confidence.

Sophistication

The interface and experience reflect the caliber of the objects and people it serves. Every detail is considered.

Precision

From cataloguing to deal structure, every tool is built for accuracy and professional-grade workflows.

Exclusivity

Access is earned. Rooms are invite-only. The platform exists to serve serious collectors and institutions.

Personality

Minimalist Sophisticated Premium Exclusive Modern Trustworthy

All elements use stroke-only rendering at 3px weight in Warm Gold (#D8B48A). The alternating pattern of rounded rectangles and circles creates a balanced, geometric mark that conveys both precision and sophistication.

Clear Space

Minimum Clear Space

The minimum clear space around the logo equals the circle radius (~22 units, approximately 14% of the logo width). This ensures the mark retains its visual integrity when placed alongside other elements.

The dashed border in the demo represents the minimum exclusion zone.

07 / Motion

Motion & Animation

Animation reinforces brand personality. All motion should feel sophisticated, deliberate, and never distracting. We use subtle transitions that guide attention without overwhelming the experience.

Timing Functions

Quick Interactions

0.2s

Hover states, focus rings, color changes. Instant feedback for micro-interactions.

Standard Transitions

0.35s

Button animations, input focus, link hovers. The default for most UI elements.

Emphasis Transitions

0.4s - 0.5s

Modal entrances, panel slides, content reveals. Deliberate, noticeable animations.

Easing Curves

ease-out

Primary easing for entrances

ease-in-out

Symmetric transitions, toggles

ease

Default, general-purpose

Animation Patterns

Fade In

Scroll-Triggered Entrance

Content fades in and translates up as it enters the viewport. Uses AOS (Animate On Scroll) library with custom delays for staggered reveals.

opacity: 0 → 1; transform: translateY(100px) → translateY(0)

Hover

Button & Card Hover

Background color transitions with subtle border changes. Cards may lift slightly with a shadow increase.

transition: all 0.35s; background: transparent → #e1bb8f

Modal

Modal Entrance

Backdrop fades in while modal slides up from below. Creates depth and draws attention to new content.

opacity: 0 → 1; transform: translateY(30px) → translateY(0); duration: 0.35s

Sticky

Sticky Label Flip

Section labels flip border-radius when becoming sticky. Quick 0.15s transition maintains responsiveness.

border-radius: 20px 20px 0 0 → 0 0 20px 20px

Reduced Motion

Accessibility Consideration

Always respect prefers-reduced-motion. When enabled, disable scroll-triggered animations, reduce transition durations to near-instant (0.01s), and keep essential state changes visible without motion.

/* Reduced motion support */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

Screen Reader Considerations

Semantic HTML

Use proper heading hierarchy (h1 → h6), landmark roles (header, main, nav, footer), and meaningful link text. Never use "click here" or "read more" alone.

Alt Text

All images must have descriptive alt text. Decorative images use alt="". Icons with meaning need aria-label or visually-hidden text.

ARIA Labels

Use aria-label for icon buttons, aria-describedby for form hints, and aria-live for dynamic content updates.

Icon Guidelines

Stroke Weight

Icons use 1px stroke weight at standard sizes. Scale stroke proportionally when icon size increases significantly.

Color Usage

Icons inherit text color via currentColor. On dark backgrounds use Gold (#e1bb8f). On light backgrounds use Dark Teal (#284045).

Alignment

Icons should be optically centered within their bounds. When paired with text, vertically align to the text baseline or center.

10 / Voice & Tone

Voice & Tone

OVITRINO speaks with quiet confidence. Our voice is knowledgeable but never condescending, exclusive but welcoming to those who belong, sophisticated without being pretentious.

Voice Principles

Confident, Not Arrogant

"Infrastructure built for serious collectors."

State capabilities directly. Let quality speak for itself without boasting.

Exclusive, Not Elitist

"Access is earned. Rooms are invite-only."

Acknowledge exclusivity as a feature, not a barrier. It protects value.

Clear, Not Simplistic

"Set your access rules. Invite members. Start showcasing."

Respect the audience's intelligence while remaining accessible.

Warm, Not Casual

"Your Rooms. Your Rules."

Direct address creates connection without sacrificing professionalism.
